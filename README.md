# Principios de Ingeniería de Software

Este repositorio contiene ejemplos prácticos en Java de los siguientes principios:

## 1. DRY (Don't Repeat Yourself)
Evita la duplicación de código mediante la reutilización de métodos.

## 2. KISS (Keep It Simple, Stupid)
Promueve mantener el código lo más simple posible.

## 3. YAGNI (You Aren’t Gonna Need It)
Indica que no se debe implementar funcionalidad que no se necesita en el momento.
